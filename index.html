Documentation

Welcome to the Phrogsploit documentation. Here you'll find information on how to get started, use the features, and understand the API (if applicable).

Getting Started

Installation: Download the latest build for your platform from the Download Section above. Currently, only Windows is available. Other builds can be found on our Discord server when they become available.

First Launch: Upon launching Phrogsploit, you'll be greeted by the main interface. Familiarize yourself with the script tabs, execution controls, and file options.

Using the Executor

Paste your script into a tab or open a script file using the "Open File" button. Use the "Execute" button to run the script. Manage multiple scripts easily with the '+' button to add new tabs.

Example script:


 local Params = {
  RepoURL = "https://raw.githubusercontent.com/luau/SynSaveInstance/main/",
  SSI = "saveinstance",
 }
 local synsaveinstance = loadstring(game:HttpGet(Params.RepoURL .. Params.SSI .. ".luau", true), Params.SSI)()
 local Options = {} -- Documentation here https://luau.github.io/UniversalSynSaveInstance/api/SynSaveInstance
 synsaveinstance(Options)

Privacy Policy
